What's with the name?

"Petitio" is a hybrid between the French word for "small" ("petit"), and the Latin word for "request" ("petitio"). This library was originally called @helperdiscord/centra, but we changed it to evolve beyond our status as a Centra fork and set out to create the smallest and fastest high-level Node HTTP wrapper available.
